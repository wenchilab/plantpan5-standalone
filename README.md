# PlantPAN 5 — Stand-alone Edition

A self-contained Docker image that runs **Promoter Analysis** and
**Multiple Promoter Analysis** locally, with no outbound network.

> **What _is_ in this image:** the PlantPAN legacy motif library
> (2,254 motifs) + slim motif → family/species metadata.
> The web UI includes per-species filtering and a Family chart
> with Donut + TreeMap views.
>
> **What is _not_ in this image:** the curated TF annotation —
> gene-ID and TF-locus mappings, ChIP-seq peaks, cross-species
> homologs, PPI tables, full TF metadata. Each result row links
> back to the online PlantPAN site (POST to TFBS_info.php) for
> users who need full annotation.

Maintained by the **PBMB Lab** (Plant Bioinformatics and Molecular
Biology Laboratory), Institute of Tropical Plant Sciences and
Microbiology, National Cheng Kung University, Tainan, Taiwan —
PI Prof. Wen-Chi Chang (<wenchilab@gmail.com>).

> **Note on the repo contents:** this repository ships *source code only*.
> The motif libraries and slim metadata are build artifacts
> (`bin/`, `data/`) generated by `build.sh` and are intentionally
> git-ignored. Distribute the runnable build via `package.sh` (zip) or a
> Docker image, not via git.

## Quick start

```bash
docker run --rm -p 8080:80 ghcr.io/wenchilab/plantpan5-offline:latest
# open http://localhost:8080
```

## Building from source

You need a working clone of the live PlantPAN 5 source tree as a sibling
directory (so `../program_base/` resolves):

```bash
cd plantpan5-offline
./build.sh                                                # produces plantpan5-offline:latest
./scripts/verify_image.sh plantpan5-offline:latest        # check no leaks
docker run --rm -p 8080:80 plantpan5-offline:latest
```

## Distributing as a self-contained zip

```bash
./scripts/package.sh                                      # produces ../plantpan5-offline-vX.Y-DATE.zip
```

Recipient unzips, runs `./build.sh`, no PlantPAN source needed.

## Limits in the offline build

- No hard cap on sequence count (effectively unlimited)
- 50,000 bp per sequence
- 100,000,000 bp aggregate per job (FASTA headers excluded)
- Input text up to 100 MB; per-job output cap 5 GB
- No MEME / TomTom / cross-species / ChIP-seq / PPI / homolog data

Need any of these? Use the live site at <https://plantpan.itps.ncku.edu.tw>.
